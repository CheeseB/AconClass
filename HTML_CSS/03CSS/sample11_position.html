position: static (디폴트)

페이지 흐름에 따라 자동적으로 위치되며, 특정 위치 설정(top/bottom/left/right)에 영향을 받지 않음

static 포지션인 div

position: relative

디폴트인 static일 때의 위치 기준으로 상대적인 위치가 결정되며, top/bottom/left/right 으로 위치를 결정함

relative 포지션인 div

position: fixed

뷰포트 기준으로 상대적인 위치가 결정되며, 페이지가 스크롤 되어도 항상 고정된 자리에 위치함

fixed 포지션인 div

position: absolute

가장 가까운 부모 기준으로 상대적인 위치가 결정되며, 만약 부모가 없거나 부모가 static 포지션이라면 body를 기준으로 설정됨 (스크롤 시 움직임)

relative 포지션인 부모 div
absolute 포지션인 div
