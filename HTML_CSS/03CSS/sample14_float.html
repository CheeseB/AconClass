float 속성

특정 태그를 left/right 로 밀어붙일 수 있음 (up/down은 불가)

float 태그의 이전 요소는 영향을 받지 않으며, clear 속성으로 해제 가능함

웹페이지의 레이아웃을 구성할 때 반드시 사용하며, 부유하는 대상을 만들 때 사용함


같은 레벨에 있는 요소에 float를 적용하면 수평 정렬됨 (기본은 수직정렬)

다음 요소부터 float를 적용하고 싶지 않으면 clear 속성으로 해제 가능


자손에게 float 속성 적용할 시 반드시 부모에게 overflow: hidden; 해줘야 함

그렇지 않으면 부모가 wrap이 한줄로 되며 레이아웃이 망가짐
